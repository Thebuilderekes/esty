# List a Place

A modern, high-conversion web platform template designed for property owners, landlords, and real estate investors looking to list and manage their rental assets seamlessly. This project features a clean, responsive layout tailored to B2B user experiences, focusing on performance metrics and value propositions that drive engagement.

## 🚀 Features

- **Owner-Centric Copywriting:** Re-engineered from typical tenant-focused landing pages to speak directly to property ROI and asset management.
- **Responsive Split-Screen Contact Layout:** Seamless transition from desktop grid view to mobile-first stacks for high-conversion form submissions.
- **Modern Off-White Design System:** Utilizing a premium color palette (`#f8fafc` / `#faf9f6`) to establish a clean, trustworthy aesthetic.
- **Component-Driven Structure:** Modular architecture featuring a scannable Stats Grid and custom CSS `feature-card` elements.
- **Zero Dependencies:** Built entirely with vanilla HTML5 and CSS3 for lightning-fast performance and clean rendering.


## 🛠️ Built With

* **HTML5:** Semantic architecture (`<main>`, `<section>`, `<aside>`) for strong SEO positioning.
* **CSS3:** Custom properties/variables, Flexbox, and CSS Grid layout techniques for ultimate responsiveness.
* **Bootstrap:** Used for navigation, sliding gallery and Tabbed content

## 🎨 UI & Design Tokens

The project relies on a modern, high-contrast palette optimized for readability and trust:

* **Primary Accent:** `#2563eb` (Vibrant Trust Blue)
* **Background Light:** `#faf9f6` / `#f8fafc` (Warm Editorial Off-White)
* **Text Dark:** `#1e293b` (Slate 800 for high readability)
* **Typography:** System-ui font stack (`-apple-system`, `BlinkMacSystemFont`, `Segoe UI`) for fast native loading without external font network overhead.

## 🔧 Getting Started

To view or modify this project locally:

1. **Clone the repository:**

```bash
   git clone https://github.com/Thebuilderekes/esty

```

2. **Navigate to the directory:**

```bash
   cd List-a-place

```

3. **Open in browser:**
Simply double-click `about.html` or run it via a local development server like VS Code's *Live Server* or a quick Python environment:

```bash
   python3 -m http.server 8000

```
